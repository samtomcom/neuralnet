# NeuralNet

A bare-bones framework for Artifical Neural Networks.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'neuralnet', :git => "https://github.com/samtomcom/neuralnet.git", :require => false
```

And then execute:

    $ bundle
